## 🧠 Features

- 🔍 Search any GitHub username
- 📊 Displays profile info (bio, followers, location, etc.)
- 🖼 Avatar and profile link
- 📅 Account creation date
- 📦 Public repository count
- 🧑‍💼 Hireable status
- 📱 Fully responsive for all devices
